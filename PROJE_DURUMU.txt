2. MODÜL DETAYLARI — NE YAPILDI?
================================================================================

--- MODÜL 1: VERİ BORU HATTI (pipeline/) ---
✅ DatabaseManager     : SQLite 3 tablo (financial_data, news_articles, nlp_scores)
                         + 1 view (daily_merged). Upsert, transaction, context manager.
✅ FinancialDataFetcher: yfinance → log return hesaplama → DB'ye kayıt.
                         XU100.IS 1 yıllık veri (250 iş günü) çekildi.
✅ MockNewsFetcher     : TCMB/FED/Yerel mock haber üretici. 525 haber üretildi.
                         Gerçek scraper'lar yeterli haber getiremediyse fallback.
✅ TCMBNewsFetcher     : TCMB basın duyuruları + PPK kararları web scraper.
                         HTML parse, tarih filtre, detay sayfa içerik çekme.
                         Retry + timeout + hata toleransı.
✅ FEDNewsFetcher      : Fed RSS feed (press_monetary.xml, press_all.xml) +
                         FOMC takvim sayfası statement scraper. XML parse,
                         duplicate URL tespiti, detay sayfa içerik çekme.
✅ GDELTNewsFetcher    : GDELT DOC 2.0 API ile Türkiye ekonomi haberleri.
                         Ücretsiz JSON API, tarih aralığı chunking (30 gün),
                         max_articles_per_day limiti, seen URL dedupe.
                         Sorgu: Turkey/TCMB/Turkish lira + economy/interest rate/
                         inflation/monetary policy/central bank/GDP/trade/fiscal.
✅ DataPreprocessor    : IQR/Z-score outlier temizleme, ffill/bfill/interpolate,
                         feature engineering (abs_return, return_sq, rolling_vol,
                         rolling_mean, uncertainty_sma, high_uncertainty).

--- MODÜL 2: LLM OLAY ÇIKARIMI (nlp/) ---
✅ prompts.py          : 13 olay türü tanımlı system prompt. JSON schema zorlayan
                         yapılandırılmış çıktı. Tek ve batch prompt builder.
✅ OllamaClient        : Ollama REST API wrapper. Sağlık kontrolü, JSON mode,
                         timeout, hata yönetimi, JSON extract fallback.
✅ MockNLP             : 33 keyword'lük ağırlıklı sözlük. 11 olay türü tespiti.
                         Ollama erişilemezken otomatik fallback.
✅ EventExtractor      : Ana orkestratör. Ollama varsa LLM, yoksa MockNLP.
                         Validate → DB kayıt → DataFrame dönüşüm → Özet istatistik.
                         525 haber analiz edildi (ort. belirsizlik: 0.55).

--- MODÜL 3: FİNANSAL MODELLEME (models/) ---
✅ GARCHEngine         : 6 model tek seferde eğitilir:
                         GARCH(1,1), EGARCH(1,1), TARCH(1,1,1)
                         GARCH-X, EGARCH-X, TARCH-X (belirsizlik skoru exog var.)
                         Forecast (5 gün ileriye), volatilite/residual DataFrame.
✅ ModelBenchmark      : AIC/BIC sıralama, Vol korelasyonu, Vol MAE,
                         Ljung-Box otokorelasyon testi, Jarque-Bera normallik testi,
                         Exog değişken anlamlılık raporu (p-value).

================================================================================

  # 1. Tam pipeline (gerçek TCMB + FED + GDELT scraper):
  python3 main.py

  # 2. Sadece mock haberle hızlı çalıştır:
  python3 main.py --mock-only --force-mock

  # 3. Farklı ticker ile:
  python3 main.py --ticker "^GSPC" --mock-only --force-mock

  # 4. Mevcut DB verisiyle (yeniden çekmeden):
  python3 main.py --skip-fetch --force-mock

  # 5. Tüm modülleri ayrı ayrı test et:
